这是一个我一万年前写的单词听写程序，最近背英语的时候把他翻出来了。

添加要听写的单词请自行在words文件创建一个文本文件。

把要听写的单词以单词空格中文，一行一个单词的格式放进去即可。具体可以康康那个Unit1.txt文件。
